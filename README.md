# Notice Ninja

Notice Ninja turns lease clauses and landlord messages into a clear notice timeline, move-out checklist, risk scan, and a sendable reply draft.

## What it does

Notice Ninja helps renters understand what they likely need to do next when a lease, renewal, or move-out notice feels confusing.

## How to Run (from zero)

### Prerequisites
- Node.js 22+
- npm 10+

### Clone
```bash
git clone https://github.com/sundaiclaw/notice-ninja.git
cd notice-ninja
```

### Install
```bash
npm install
```

### Run
```bash
npm run start
```

Open: `http://localhost:8080`
